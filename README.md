# Domus Spectra v1.0

Domus Spectra main repository. Here you will find all necessary data to build up our project in your PC.

## Requirements

-Cocos2d-x v-3.17

## Installation

Run inside the folder

```shell
~$ cocos compile -p linux
```

Then, go to *./bin/debug/linux* folder and execute.

```shell
~$ ./DomusSpectra
```
